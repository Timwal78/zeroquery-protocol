The Problem

You gave your AI agent a live wallet. Now it's drained.

Today's autonomous AI systems execute trades and intents on-chain with blind trust. Agents are ephemeral bursts of computation, yet they hold persistent liquidity or rely on trusted Web2 APIs. The infrastructure hasn't caught up.

Why this matters:

  • Relying on off-chain execution agents requires trusting them with funds.
  • API data provided by agents can be trivially spoofed.
  • There is no cryptographic proof linking an agent's off-chain action to an on-chain settlement.
  • A rogue agent or exploited API key compromises the entire liquidity pool.

The Solution

ZeroQuery: Cryptographic provenance for AI intents.

ZeroQuery acts as an off-chain gossip relay paired with an on-chain Zero-Knowledge verifier. Instead of trusting an agent to execute correctly, you escrow your funds and force them to prove it.

How it works:

  • Gossip Network: Broadcast your intent (e.g., "Buy SOL if Price < $150") to the decentralized ZeroQuery Relay.
  • IntentRank: The network mathematically ranks agents based on their cross-chain DID reputation on the Xahau ledger.
  • Layer 5 ZK-SNARKs: The winning agent executes the API call inside a Succinct SP1 zkVM, generating a cryptographic proof of the execution.
  • Trustless Settlement: The agent submits the proof to the Solana poi-verifier smart contract. If valid, the Escrow releases the bounty.
